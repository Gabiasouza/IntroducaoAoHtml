Lista de Tags HTML e Suas Funcionalidades

Tag Descrição Funcionalidade
<!DOCTYPE> Declara o tipo de documento Define o documento como HTML5
<html></html> Define o documento HTML Envolve todo o conteúdo da página
<head></head> Contém metadados sobre o documento Inclui título, links para estilos, scripts, etc.
<title></title> Define o título da página Exibido na aba do navegador
<body></body> Define o corpo do documento Contém todo o conteúdo visível da página
<h1> a <h6> Tags de cabeçalho Definem títulos e subtítulos
<p></p> Parágrafo Define blocos de texto
<a></a> Âncora (link) Cria hiperlinks para outras páginas ou seções
<img> Imagem Insere uma imagem no documento
<ul></ul> Lista não ordenada Cria uma lista com marcadores
<ol></ol> Lista ordenada Cria uma lista numerada
<li></li> Item de lista Define itens em listas ordenadas ou não ordenadas
<table></table> Tabela Cria uma estrutura de tabela
<tr></tr> Linha da tabela Define uma linha em uma tabela
<td></td> Célula da tabela Define uma célula de dados
<th></th> Cabeçalho da tabela Define uma célula de cabeçalho
<form></form> Formulário Cria um formulário de entrada de dados
<input> Entrada de dados Define campos de entrada para formulários
<button></button> Botão Cria um botão clicável
<div></div> Divisão Define uma seção de bloco
<span></span> Span Define uma seção em linha
<footer></footer> Rodapé Define o rodapé da página
<header></header> Cabeçalho Define o cabeçalho da página
<section></section> Seção Define uma seção de conteúdo
<article></article> Artigo Define um conteúdo independente e autônomo
<nav></nav> Navegação Define links de navegação
